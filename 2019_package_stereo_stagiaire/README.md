https://www.python.org/downloads/
*penser à cocher add to path pour utiliser la ligne de commande*

python -m pip install --upgrade pip
```

pip install PyQt5
pip install pygame

```
## Lancer le programme
Se placer à la racine du dossier
```bash
python main.py
```
